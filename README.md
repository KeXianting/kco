# 目录结构说明
由于之前没有怎么整理项目,是的结构十分混乱,因此抽空把所用的项目给整理了一下

- kco  
    - example               以下所有项目的代码都集合在这里
        - src/python        存放python的程序
        - src/main/java/com/kco
            - game          存放一下小游戏的程序
                - maze      迷宫的生成
            - primarykeybuild 自定义主键生成策略
            - profit        两个工厂模式实现分润规则
            - searchfile    用lucene实现文件索引的简单例子
            - secret        密码学
            - thread        多线程专题
    - SimpleProgram         存放小程序 
    - php                   php自我测试程序
    - primaryKeyBuild       自定义主键生成策略
    - pythontest            python的测试程序
    - searchFile            用lucene实现文件索引的简单例子
    - threadTest            多线程专题
